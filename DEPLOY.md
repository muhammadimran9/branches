# PakBizBranches - Pakistan Business Directory

## 🚀 Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/muhammadimran9/branches)

### Quick Deploy Steps:

1. **Click "Deploy" button above** or go to [Vercel](https://vercel.com)
2. **Import Repository**: `muhammadimran9/branches`
3. **Configure Environment Variables** (Optional):
   - Add Firebase credentials if using backend features
   - See `.env.example` for required variables
4. **Click Deploy**

### Local Development:

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build

# Start production server
pnpm start
```

Open [http://localhost:3000](http://localhost:3000)

## 📦 Tech Stack

- **Framework**: Next.js 16
- **UI**: Tailwind CSS + Radix UI
- **Database**: Firebase Firestore
- **Storage**: Firebase Storage
- **Deployment**: Vercel

## 🔧 Configuration

### Firebase Setup (Optional):
1. Create Firebase project
2. Enable Firestore & Storage
3. Add credentials to Vercel environment variables

### Custom Domain:
1. Go to Vercel Dashboard → Settings → Domains
2. Add: `pakbizbranhces.online`
3. Update DNS records
